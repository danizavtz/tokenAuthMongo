# tokenAuthMongo
generate jwt token and persist in mongodb

## Instructions to run

1. Copy a env-sample to .env
`$ cp env-sample .env
2. Fill in the variables with your environment configuration
3. Seed the database
`$ npm run seeddb`
3. Just start the server
`$npm run start`
